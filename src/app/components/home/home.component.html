HelpDesk Introdução

Olá! Bem-vindo ao sistema de Help Desk que desenvolvi durante o curso de Desenvolvedor Full Stack.
Este é um sistema de gerenciamento de chamados, simulando um Help Desk simples criado com Angular 12 e Spring Boot.
O projeto tem como objetivo principal abordar conceitos como Single Page Applications (SPAs), Autenticação e Autorização com Tokens JWT, e muito mais.
O código-fonte do projeto pode ser encontrado no Github, basta clicar no link no menu lateral.

Sinta-se à vontade para testar o sistema e deixar suas sugestões.
